# 套料优化算法设计文档

## 1. 问题建模

### 1.1 问题定义
这是一个**一维下料问题（1D Cutting Stock Problem）**，目标是最大化材料利用率。

### 1.2 数学模型

**集合定义**：
- J = {1, 2, ..., n}：零件集合
- L = {l₁, l₂, ..., lₘ}：可用原材料长度集合
- S = {1, 2, ..., p}：规格分组

**参数定义**：
- dⱼ：零件j的需求量
- lenⱼ：零件j的长度
- Lₖ：原材料长度k
- hₛ：规格s的头尾损耗
- wₛ：规格s的单刀损耗
- Mⱼ：大数常数（零件j的最大可能切割数）

**决策变量**：
- xₖ,ᵢ,ⱼ ∈ Z⁺：第i根长度k的原材料上切割零件j的数量
- yₖ,ᵢ ∈ {0,1}：是否使用第i根长度k的原材料
- zₖ,ᵢ,ⱼ ∈ {0,1}：第i根长度k的原材料上是否切割零件j
- rₖ,ᵢ ∈ R⁺：第i根长度k的原材料的余料

**目标函数**（最小化原材料总长度 + 余料惩罚）：
```
min  Σₖ,ᵢ (yₖ,ᵢ × Lₖ + α × rₖ,ᵢ)
```
其中 α 是惩罚权重（默认0.1）

**约束条件**：

1. **需求满足约束**：
```
Σₖ,ᵢ xₖ,ᵢ,ⱼ = dⱼ    ∀j ∈ J
```

2. **容量约束**（含损耗）：
```
Σⱼ (xₖ,ᵢ,ⱼ × lenⱼ) + hₛ + wₛ × Σⱼ zₖ,ᵢ,ⱼ ≤ Lₖ × yₖ,ᵢ    ∀k, i
```

3. **零件号约束（材料侧）**：
```
Σⱼ zₖ,ᵢ,ⱼ ≤ 3    ∀k, i
```

4. **零件号约束（零件侧）**：
```
Σₖ,ᵢ zₖ,ᵢ,ⱼ ≤ 3    ∀j ∈ J
```

5. **关联约束**：
```
xₖ,ᵢ,ⱼ ≤ Mⱼ × zₖ,ᵢ,ⱼ    ∀k, i, j
zₖ,ᵢ,ⱼ ≤ yₖ,ᵢ           ∀k, i, j
```

6. **余料定义约束**：
```
rₖ,ᵢ = Lₖ × yₖ,ᵢ - Σⱼ (xₖ,ᵢ,ⱼ × lenⱼ) - hₛ × yₖ,ᵢ - wₛ × Σⱼ zₖ,ᵢ,ⱼ    ∀k, i
```

7. **对称性破除**：
```
yₖ,ᵢ ≥ yₖ,ᵢ₊₁    ∀k, i
```

### 1.3 关键设计决策

#### 1.3.1 目标函数转换
由于零件总长度固定，最大化利用率等价于**最小化原材料总长度**。这将分式目标函数转化为线性目标函数。

#### 1.3.2 余料作为软约束
余料约束作为惩罚项加入目标函数，而非硬约束。原因：
- 硬约束可能导致无可行解
- 某些场景下余料超过1m是可接受的
- 通过惩罚项平衡利用率和余料控制

#### 1.3.3 按规格分组独立求解
不同规格的零件不能混用原材料，分组求解可以：
- 大幅减少问题规模
- 并行处理（未来扩展）
- 更好的可解释性

### 2.2 MIP求解器

#### 2.2.1 求解器选择
使用 **Google OR-Tools CBC** 求解器：
- 开源免费
- 性能良好
- 易于集成

#### 2.2.2 模型实现要点

```python
from ortools.linear_solver import pywraplp

def build_mip_model(parts, materials, loss_rule, config):
    solver = pywraplp.Solver.CreateSolver('CBC')
    solver.SetTimeLimit(config.time_limit * 1000)

    # 1. 创建变量
    x = {}  # 切割数量
    y = {}  # 使用材料
    z = {}  # 零件是否在材料上
    r = {}  # 余料

    for length in lengths:
        for i in range(max_materials):
            y[length, i] = solver.IntVar(0, 1, f'y_{length}_{i}')
            r[length, i] = solver.NumVar(0, length, f'r_{length}_{i}')

            for j, part in enumerate(parts):
                x[length, i, j] = solver.IntVar(0, part.quantity, f'x_{length}_{i}_{j}')
                z[length, i, j] = solver.IntVar(0, 1, f'z_{length}_{i}_{j}')

    # 2. 添加约束
    add_demand_constraints(solver, x, parts, lengths, max_materials)
    add_capacity_constraints(solver, x, y, z, parts, lengths, max_materials, loss_rule)
    add_part_count_constraints(solver, z, parts, lengths, max_materials, config)
    add_material_count_constraints(solver, z, parts, lengths, max_materials, config)
    add_linking_constraints(solver, x, y, z, parts, lengths, max_materials)
    add_remainder_constraints(solver, r, x, y, z, parts, lengths, max_materials, loss_rule)
    add_symmetry_breaking_constraints(solver, y, lengths, max_materials)

    # 3. 设置目标函数
    objective = solver.Objective()
    for length in lengths:
        for i in range(max_materials):
            objective.SetCoefficient(y[length, i], length)
            objective.SetCoefficient(r[length, i], 0.1)  # 余料惩罚
    objective.SetMinimization()

    return solver, x, y, z, r
```

#### 2.2.3 关键约束实现

**容量约束（含损耗）**：
```python
def add_capacity_constraints(solver, x, y, z, parts, lengths, max_materials, loss_rule):
    for length in lengths:
        for i in range(max_materials):
            # Σ(x * len) + head_tail + single_cut * Σz <= L * y
            ct = solver.Constraint(-solver.infinity(), length)
            
            # 零件长度
            for j, part in enumerate(parts):
                ct.SetCoefficient(x[length, i, j], part.length)
            
            # 刀损耗
            for j in range(len(parts)):
                ct.SetCoefficient(z[length, i, j], loss_rule.single_cut_loss)
            
            # y变量
            ct.SetCoefficient(y[length, i], -length + loss_rule.head_tail_loss)
```

**对称性破除**：
```python
def add_symmetry_breaking_constraints(solver, y, lengths, max_materials):
    # 相同长度的材料按顺序使用：y[0] >= y[1] >= y[2] ...
    for length in lengths:
        for i in range(max_materials - 1):
            ct = solver.Constraint(-solver.infinity(), 0)
            ct.SetCoefficient(y[length, i], 1)
            ct.SetCoefficient(y[length, i + 1], -1)
```

### 2.3 贪心算法（回退方案）

#### 2.3.1 算法流程

```
贪心套料算法：

输入：零件列表、原材料列表、损耗规则、配置
输出：切割方案列表

1. 初始化：
   - 复制零件列表（带剩余数量）
   - 获取可用原材料长度（升序）

2. 主循环：
   while 存在未满足的零件需求:
       a. 对每种原材料长度：
          - 贪心填充零件
          - 计算利用率
       
       b. 选择利用率最高的材料
       
       c. 更新剩余零件需求
       
       d. 记录切割方案

3. 返回切割方案列表

贪心填充单根材料：
1. 按长度降序排列零件
2. 依次尝试加入零件：
   - 检查零件号限制（≤3种）
   - 检查容量限制
   - 加入尽可能多的该零件
3. 返回填充结果
```

#### 2.3.2 核心实现

```python
class GreedyNestingSolver:
    def solve(self, parts, materials, spec, material_type):
        loss_rule = self.loss_calculator.get_loss_rule(spec, material_type)
        remaining_parts = [(p.part_no, p.length, p.quantity) for p in parts]
        available_lengths = sorted(set(m.length for m in materials))
        length_to_material = {m.length: m for m in materials}
        
        cutting_plans = []
        
        while any(p[2] > 0 for p in remaining_parts):
            active_parts = [p for p in remaining_parts if p[2] > 0]
            
            # 选择最优材料
            best_plan = None
            best_utilization = -1
            
            for length in available_lengths:
                raw_mat = length_to_material[length]
                plan = self._fill_material(raw_mat, active_parts, loss_rule)
                
                if plan and plan.utilization > best_utilization:
                    best_utilization = plan.utilization
                    best_plan = plan
            
            # 更新剩余需求
            for part_no, part_length, qty in best_plan.parts:
                for i, (p_no, p_len, p_qty) in enumerate(remaining_parts):
                    if p_no == part_no and p_len == part_length:
                        remaining_parts[i] = (p_no, p_len, p_qty - qty)
                        break
            
            cutting_plans.append(best_plan)
        
        return cutting_plans

    def _fill_material(self, raw_material, parts, loss_rule):
        available = raw_material.length - loss_rule.head_tail_loss
        sorted_parts = sorted(parts, key=lambda x: x[1], reverse=True)
        
        selected = []
        part_no_set = set()
        
        for part_no, part_length, remaining_qty in sorted_parts:
            if remaining_qty <= 0:
                continue
            
            # 检查零件号限制
            if len(part_no_set) >= self.config.max_parts_per_material:
                if part_no not in part_no_set:
                    continue
            
            # 计算可加入数量
            current_len = sum(p[1] * p[2] for p in selected)
            max_qty = min(
                remaining_qty,
                (available - current_len) // part_length
            )
            
            if max_qty > 0:
                selected.append((part_no, part_length, int(max_qty)))
                part_no_set.add(part_no)
        
        if not selected:
            return None
        
        # 构建切割方案
        cut_count = len(selected)
        used_length = sum(p[1] * p[2] for p in selected)
        total_loss = loss_rule.head_tail_loss + loss_rule.single_cut_loss * cut_count
        
        return CuttingPlan(
            raw_material=raw_material,
            parts=selected,
            cut_count=cut_count,
            single_cut_loss=loss_rule.single_cut_loss,
            head_tail_loss=loss_rule.head_tail_loss,
            used_length=used_length,
            total_loss=total_loss,
            remaining_length=raw_material.length - used_length - total_loss,
            utilization=used_length / raw_material.length
        )
```

### 2.4 损耗计算器

```python
class LossCalculator:
    def get_loss_rule(self, spec: str, material: str) -> LossRule:
        """根据规格和材质获取损耗规则"""
        limb_width, thickness = self._parse_spec(spec)
        
        for rule in self.loss_rules:
            # 检查肢宽范围
            if not (rule.limb_width_range[0] <= limb_width <= rule.limb_width_range[1]):
                continue
            
            # 检查厚度范围
            if rule.thickness_range != (0, 999):
                if not (rule.thickness_range[0] <= thickness <= rule.thickness_range[1]):
                    continue
            
            # 检查材质
            if rule.materials and material not in rule.materials:
                continue
            
            return rule
        
        # 默认规则
        return self.loss_rules[0]

    @staticmethod
    def _parse_spec(spec: str) -> Tuple[int, int]:
        """解析规格 L90X7 -> (90, 7)"""
        try:
            parts = spec.upper()[1:].split('X')
            return int(parts[0]), int(parts[1])
        except:
            return 0, 0
```

## 3. 性能优化

### 3.1 变量数量估计

材料数量上界估计：
```python
def _estimate_max_materials(self, parts, materials, loss_rule):
    total_part_length = sum(p.length * p.quantity for p in parts)
    min_material_length = min(m.length for m in materials)
    estimated = total_part_length / (min_material_length * 0.8)
    return max(int(estimated * 1.5), len(parts) * 2, 10)
```

### 3.2 对称性破除

相同长度的材料按顺序使用，减少搜索空间：

## 4. 算法复杂度

### 4.1 MIP求解
- 变量数：O(|L| × M × |J|)，其中M是估计的材料数上界
- 约束数：O(|L| × M × |J|)
- 最坏情况：指数时间
- 实际情况：中等规模（<50零件）通常在60秒内求解

### 4.2 贪心算法
- 时间复杂度：O(n × m × log(n))，其中n是零件数，m是材料数
- 空间复杂度：O(n + m)
- 适合大规模问题

```python
def validate_result(result: NestingResult, parts: List[Part], config: NestingConfig):
    # 1. 检查需求满足
    part_usage = defaultdict(int)
    for plan in result.cutting_plans:
        for part_no, length, qty in plan.parts:
            part_usage[(part_no, length)] += qty
    
    for part in parts:
        key = (part.part_no, part.length)
        assert part_usage[key] >= part.quantity, f"零件 {part.part_no} 需求未满足"
    
    # 2. 检查约束
    for plan in result.cutting_plans:
        assert plan.cut_count <= config.max_parts_per_material, "零件号超限"
    
    # 3. 计算实际利用率
    total_used = sum(plan.used_length for plan in result.cutting_plans)
    total_material = sum(plan.raw_material.length for plan in result.cutting_plans)
    actual_utilization = total_used / total_material
    
    assert abs(actual_utilization - result.total_utilization) < 0.01, "利用率计算错误"
```
