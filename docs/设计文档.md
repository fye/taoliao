# 套料优化系统设计文档

## 1. 项目概述

### 1.1 背景
电力行业中，角钢套料是一个典型的下料优化问题（Cutting Stock Problem）。需要在满足各种约束条件下，将零件合理分配到原材料上，以最大化材料利用率、降低成本。

### 1.2 目标
- 最大化材料利用率
- 满足操作便利性约束
- 生成可执行的套料方案

## 2. 需求分析

### 2.1 输入数据
| 数据类型 | 文件 | 关键字段 |
|---------|------|---------|
| 需求清单 | 需求清单.xlsx | 部件号、材质、规格、长度(mm)、单基数量(件) |
| 市场清单 | 角钢市场清单.xlsx | 材质、规格全称、长度、市场货存量 |
| 损耗规则 | 损耗规则.xlsx | 肢宽范围、厚度范围、材质、单刀损耗、头尾损耗 |

### 2.2 核心约束
| 约束名称 | 默认值 | 说明 | 约束类型 |
|---------|-------|------|---------|
| 单根原材料零件号上限 | 3 | 1根原材料最多套3个不同零件号 | 硬约束 |
| 单零件号原材料上限 | 5 | 每个零件号最多配到5根原材料 | 硬约束 |
| 余料上限 | 1000mm | 每根原材料余料不超过1米 | 软约束（惩罚项） |

### 2.3 损耗计算规则
```
总损耗 = 单刀损耗 × 切割刀数 + 头尾损耗
余料 = 母料长度 - 零件总长度 - 总损耗
利用率 = 零件总长度 / 母料长度
```

损耗规则表：
| 肢宽范围 | 厚度范围 | 材质 | 单刀损耗(mm) | 头尾损耗(mm) |
|---------|---------|------|-------------|-------------|
| L40-L56 | 不限 | 不限 | 10 | 30 |
| L63-L75 | 不限 | 不限 | 0 | 10 |
| L80-L90 | 不限 | 不限 | 15 | 35 |
| L100-L180 | ≤12 | Q235/Q355/Q420 | 20 | 55 |
| L140及以上 | ≥14 | Q235/Q355/Q420 | 2 | 8 |
| 不限 | 不限 | Q460 | 2 | 8 |

### 2.4 输出格式
生成Excel文件，包含3个Sheet：
1. **原始需求清单**：直接复制输入的需求清单
2. **套料结果**：详细的每根原材料使用情况
3. **原材料汇总**：按材质、规格统计原材料使用量

## 4. 数据模型设计

### 4.1 核心数据类

```python
from dataclasses import dataclass
from typing import List, Optional, Tuple

@dataclass
class Part:
    """零件信息"""
    part_no: str           # 部件号
    material: str          # 材质 (如 Q235B, Q355B)
    spec: str              # 规格 (如 L90X7)
    length: int            # 长度(mm)
    quantity: int          # 需求数量
    width: Optional[int]   # 宽度(mm)
    weight: Optional[float] # 单件重量(kg)
    holes: Optional[int]   # 单件孔数
    remark: Optional[str]  # 备注
    segment_no: Optional[str]  # 段号

@dataclass
class RawMaterial:
    """原材料信息"""
    material_type: str     # 材质
    spec: str              # 规格 (如 L100X10)
    length: int            # 长度(mm)
    stock: int             # 市场货存量

@dataclass
class LossRule:
    """损耗规则"""
    limb_width_range: Tuple[int, int]   # 肢宽范围 (min, max)
    thickness_range: Tuple[int, int]    # 厚度范围 (min, max)
    materials: List[str]                # 适用材质
    single_cut_loss: int                # 单刀损耗(mm)
    head_tail_loss: int                 # 头尾损耗(mm)

@dataclass
class CuttingPlan:
    """单根原材料的切割方案"""
    raw_material: RawMaterial
    parts: List[Tuple[str, int, int]]  # [(部件号, 长度, 数量), ...]
    cut_count: int                      # 切割刀数
    single_cut_loss: int                # 单刀损耗
    head_tail_loss: int                 # 头尾损耗
    used_length: int                    # 使用长度
    total_loss: int                     # 总损耗
    remaining_length: int               # 剩余长度
    utilization: float                  # 利用率

@dataclass
class NestingResult:
    """套料结果"""
    original_parts: List[Part]
    cutting_plans: List[CuttingPlan]
    material_summary: Dict[Tuple[str, str], Dict]

@dataclass
class NestingConfig:
    """套料配置参数"""
    max_parts_per_material: int = 3      # 单根原材料零件号上限
    max_materials_per_part: int = 5      # 单零件号原材料上限
    max_remainder: int = 1000            # 余料上限(mm)
    time_limit: int = 3600               # 求解时间限制(秒)
```

## 5. 核心流程

### 5.1 主流程

```
1. 加载数据
   ├── 加载需求清单 (parts)
   ├── 加载市场清单 (materials)
   └── 加载损耗规则 (loss_rules)

2. 预处理
   ├── 按材质+规格分组零件
   └── 筛选每组的可用原材料

3. 优化求解
   └── 对每个分组独立求解：
       ├── 尝试MIP精确求解
       │   ├── 成功 → 返回切割方案
       │   └── 失败/超时 → 回退贪心算法
       └── 返回切割方案

4. 汇总结果
   ├── 合并所有分组的切割方案
   ├── 计算材料使用汇总
   └── 计算总利用率

5. 导出结果
   ├── Sheet 1: 原始需求清单
   ├── Sheet 2: 套料结果
   └── Sheet 3: 原材料汇总
```

### 5.2 求解策略选择

```
判断条件：
├── 零件数 ≤ 30 且 求解时间 ≤ time_limit
│   └── 使用 MIP 精确求解
└── 否则
    └── 直接使用贪心算法

MIP求解状态：
├── OPTIMAL → 使用最优解
├── FEASIBLE → 使用可行解
└── NOT_SOLVED / INFEASIBLE → 回退贪心算法
```

## 6. 接口设计

### 6.1 CLI 接口

**Python版本:**
```bash
# 基本用法
cd taoliao_python
python main.py

# 完整参数
python main.py --demand ../docs/需求清单.xlsx \
               --market ../docs/角钢市场清单.xlsx \
               --output ../output/result.xlsx
```

**C#版本:**
```bash

cd taoliao_csharp
dotnet run --project Taoliao.CLI

# 完整参数
dotnet run --project Taoliao.CLI -- \
    -d ../docs/需求清单.xlsx \
    -o ../output/result.xlsx
```

### 6.2 Python API

```python
from taoliao.core import NestingOptimizer, NestingConfig
from taoliao.data import DataLoader, ResultExporter

# 加载数据
loader = DataLoader()
parts = loader.load_parts('../docs/需求清单.xlsx')
materials = loader.load_materials('../docs/角钢市场清单.xlsx')

# 创建优化器
config = NestingConfig(
    max_parts_per_material=3,
    max_materials_per_part=5,
    max_remainder=1000,
    time_limit=60
)
optimizer = NestingOptimizer(config)

# 执行优化
result = optimizer.optimize(parts, materials)

# 导出结果
exporter = ResultExporter(result)
exporter.export('../output/result.xlsx', loader.get_raw_parts_df())
```

## 7. 错误处理

### 7.1 数据校验
- 检查必要字段是否存在
- 检查数值是否合法（长度>0，数量>0）
- 检查规格格式是否正确

### 7.2 求解失败处理
- MIP超时：自动回退贪心算法
- 无可行解：回退贪心算法并警告
- 材质不匹配：使用同规格其他材质并警告

### 7.3 输出验证
- 检查所有零件需求是否满足
- 检查约束是否满足
- 计算实际利用率与报告一致
