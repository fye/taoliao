## 套料目标
- 需要在满足约束的情况下，尽可能的提高套料的占用率
- 占用率计算公式：零件长度/母料长度=利用率

## 套料规则
- 角钢 LAAXBB, AA为肢宽, BB为厚度。比如: L100X10, 100 为肢宽， 10 为厚度。
- 每种肢宽，在市场上有不同的材料长度可以套。具体可参加 角钢市场清单.xlsx。比如：角钢型号 L100X10 材质为 Q235B 的角钢，市场上有 10000、10500、11000、11500、6500、7000、7500、8000、8500、9000、9500 这些长度可以套料。
- 1 根原材料最多只能套 3 个零件号。这行规定也是为了师傅操作方便，尽量不要一根原材料配多个零件号，导致操作复杂。
- 每个零件号原则上最多只能配到 3 跟原材料上。 除非是这个零件号很多个，需求量一定超过 3 个，那只能配，否则不建议超过 3 跟原材料，这么规定是为了套料师傅方便，针对某个零件号能就近使用原材料，全部套完。
- 对于某个角钢型号，比如 LAAXBB，会存在多个材质的场景，当某个材料缺失时，可以根据 角钢市场清单.xlsx 中的说明，选择其他材质的材料。
- 对于每根原始材料的余料，建议不要超过 1M。 余料计算公式：（余料=母料-零件长度-刀损-头尾损）。
- 以上提到的 1 跟原材料最多套 3 个零件号，每个零件号原则上最多只能配到 3 个原材料，余料不超过 1M，这些信息我建议可作为参数传递到算法中，方便进行微调。如果不传，就用这些默认值。

## 需求清单
- 用户给与一个需求清单，比如 需求清单.xlsx，里面会列出来需要的零部件，每个零部件的长度，每个零部件的个数。然后根据上述的套料规则，计算出每个零部件的套料长度，生成一份最优的套料方案。

## 结果输出
- 用户会提供一批具体的角钢零件号需求清单，请根据上述的约束和清单要求，生成一份最优的套料方案。
- 套料方案包含 3 部分内容，也就是 3 个 sheet 页：
  - 第 1 个 sheet 页：原始的角钢零件号需求清单
  - 第 2 个 sheet 页：套料结果
    - 序号	原材料材质	规格	原材料长度	切割的部件号	切割刀数	单刀损	两头损耗	使用长度	剩余长度	利用率	损耗比	备注
    - 其中切割的部件号为这个材料一共切了几个零部件，每个零部件的长度，每个零部件的个数。例如：915/1049*1 + 411/996*4 + 416/816*1。915、411等表示零部件号，1049 等表示每个零部件的长度，1、4、1 表示每个零部件的个数。
  - 第 3 个 sheet 页：使用原材料结果：
    - 显示列数: 材质	规格  长度	所需数量	利用率	损耗比
